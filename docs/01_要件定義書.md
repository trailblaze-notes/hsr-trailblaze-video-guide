# 要件定義書 — 崩壊：スターレイル 開拓クエスト 動画ガイド

作成日: 2026-07-18 / 最終更新: 2026-07-19 / 作成: Claude（AI 駆動開発）

## 1. 背景・目的

『崩壊：スターレイル』（HoYoverse）の公式 YouTube チャンネルには PV・キャラクターPV・楽曲MV 等の動画が多数あるが、
**ストーリー後半の内容を含む動画をうっかり見るとネタバレを踏む**。
本サイトは開拓クエスト（メインストーリー）の進行度＝バージョンを軸に、
**「そのバージョンが来る前に見る動画」と「そのバージョンを終えた後に見る動画」**を整理して表示し、
ネタバレを回避しながら公式コンテンツを最大限楽しめるようにする。

## 2. 利用者・利用シーン

| 利用者 | シーン |
|---|---|
| プレイ中のゲーマー（本人・一般公開） | 自分の進行度のバージョンを開き、安全に見られる動画を確認する |
| クリア済みの人 | 種別ページ（キャラPV・紀行PV・曲）で全動画を一気見する |

## 3. 機能要件（実装済み）

- **F-1 バージョン一覧**: トップにメジャーバージョン（Ver.1.x〜4.x）のカードを編名付きで表示
- **F-2 バージョンページ**: メジャーバージョンごとに1ページ。マイナーバージョンを横並びチップ（路線図風）で選択すると動画リストがプルダウン展開
- **F-3 視聴タイミング分類**: 各バージョン内で「来る前に見る」（種別ごとのセクション）と「本編クリア後推奨」（入れ子折りたたみ・初期閉）を区別
- **F-4 種別ページ**: キャラクターPV / 紀行PV・ストーリー / 曲 の単独ページ。メジャーバージョンのトグルのみで区切った公開日順の一気見（ネタバレ配慮なし・トップに注意書き）
- **F-5 動画情報**: 埋め込みプレイヤー（再生可能）・種別バッジ・公開日・YouTube リンク
- **F-6 データ駆動**: バージョン・動画は JSON の追加のみで増やせる（コード変更不要）
- **F-7 ネタバレ安全側デフォルト**: 全折りたたみは初期閉。JS 無効環境でも先の動画が見えない

## 4. 非機能要件

| 項目 | 要件 |
|---|---|
| コスト | ランニングコスト 0 円（GitHub Pages のみ） |
| 性能 | 静的配信。埋め込み iframe は `loading="lazy"` |
| 対応環境 | モバイル / デスクトップのモダンブラウザ |
| 運用 | データ追加は JSON 編集 + git push のみ。デプロイは自動 |
| 保守 | 依存は astro + zod のみ |

## 5. コンテンツポリシー・規約コンプライアンス（厳格調査済み）

調査ソース: 崩スタ公式 二次創作ガイドライン（HoYoLAB article 1142555）/ YouTube 利用規約。

- 掲載する動画は**公式日本語チャンネル（@Houkaistarrail_jp）のもののみ**。転載・実況・非公式動画は掲載しない
- **公式素材の直接利用禁止**に対応: サムネイル画像の直リンク（i.ytimg.com）・ロゴ・公式アート・スクリーンショットは一切使用しない
- 動画表示は **YouTube 公式埋め込みプレイヤー**（プライバシー強化モード youtube-nocookie.com）のみ。YouTube 規約が第三者サイトでの埋め込みを明示許諾している
- フッターに「非公式ファンサイト」「権利は COGNOSPHERE PTE. LTD. に帰属」を明記
- 非営利個人運営・広告なしを維持（公式二次創作ガイドラインの許諾範囲）
- 調査でも yt-dlp 等の自動化ツールは使わない（YouTube 規約の自動アクセス制限）。通常閲覧・wiki 参照・oEmbed（公開 API）のみ

## 6. スコープ外

- ユーザー登録・サーバーサイド処理（完全静的サイト）
- 動画コンテンツの制作
- 進行度の保存機能（レビューで不要と判断し撤去）
