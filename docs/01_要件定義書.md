# 要件定義書 — ゲーム進行度別 公式動画ガイド

作成日: 2026-07-18 / 作成: Claude（AI 駆動開発）

## 1. 背景・目的

ゲームの公式 YouTube チャンネルには PV・キャラクター紹介・開発者インタビュー等の動画が多数あるが、
**ストーリー後半の内容を含む動画をうっかり見るとネタバレを踏む**。
本サイトは「自分がどこまで進めたか」を選ぶだけで、**そこまでの進行度で安全に見られる公式動画だけ**を一覧表示し、
ネタバレを回避しながら公式コンテンツを楽しめるようにする。

## 2. 利用者・利用シーン

| 利用者 | シーン |
|---|---|
| プレイ中のゲーマー（本人・一般公開） | プレイの合間に「今見ても大丈夫な動画」を探す |
| クリア済みの人 | 全動画を時系列で振り返る |

## 3. 機能要件

### 必須（MVP）

- **F-1 ゲーム一覧**: トップページに登録ゲームをカード表示する
- **F-2 進行度選択**: ゲーム詳細ページで自分の進行度（チェックポイント）を選択できる
- **F-3 動画フィルタ**: 選択した進行度以下の動画のみ表示する。先の動画は件数のみ示し、タイトル・サムネイルは隠す（ネタバレ防止）
- **F-4 動画情報**: タイトル / サムネイル / 種別（PV・紹介・インタビュー等）/ おすすめ理由 / YouTube リンクを表示する
- **F-5 進行度の記憶**: 選択した進行度をブラウザ（localStorage）に保存し、再訪時に復元する
- **F-6 データ駆動**: ゲーム・動画は JSON データの追加のみで増やせる（コード変更不要）

### 任意（将来）

- 動画のサイト内埋め込み再生（YouTube iframe）
- クリア後限定動画のセクション
- 検索・タグ絞り込み

## 4. 非機能要件

| 項目 | 要件 |
|---|---|
| コスト | ランニングコスト 0 円（無料ホスティングのみ） |
| 性能 | 静的配信。JS は進行度フィルタの最小限のみ |
| 対応環境 | モバイル / デスクトップのモダンブラウザ |
| 運用 | データ追加は JSON 編集 + git push のみ。デプロイは自動 |
| 保守 | 依存パッケージ最小限。年数回の更新でも壊れない構成 |

## 5. コンテンツポリシー

- 掲載する動画は**ゲーム公式チャンネル（またはパブリッシャー公式）のもののみ**
- 転載動画・実況動画は掲載しない
- 動画そのものはホストせず、YouTube へのリンク・埋め込みのみ（規約準拠）
- 業務・顧客に関する情報は一切含めない（公開リポジトリ前提）

## 6. スコープ外

- ユーザー登録・サーバーサイド処理（完全静的サイトとする）
- 動画コンテンツの制作
- 非公式動画のキュレーション
