# 崩壊：スターレイル 開拓クエスト 動画ガイド

『崩壊：スターレイル』のバージョン（開拓クエスト進行度）ごとに、**ネタバレを踏まずに見られる公式 YouTube 動画**を整理した非公式ファンサイト。
要件定義から設計・実装・データ調査・運用まで AI 駆動開発（Claude Code）で構築。

## 何ができるか

- メジャーバージョン（Ver.1.x〜4.x、編名付き）ごとのページで、マイナーバージョンを路線図風のチップで選ぶと動画リストが展開される
- 各バージョンで **「来る前に見る動画」** と **「本編クリア後推奨の動画」** を区別して表示
- 種別ページ（キャラクターPV / 紀行PV・ストーリー / 曲）で公開日順の一気見もできる
- 動画は公式埋め込みプレイヤーでそのまま再生可能

## データの信頼性

- バージョン切替日（全29バージョン）は公式プレス・攻略 wiki の2ソースで突合済み
- 全動画の youtubeId は YouTube oEmbed で「実在・公式チャンネル・タイトル一致」を機械検証済み
- 「掲載バージョンより先の動画が混ざらない」ことを Zod がビルド時に機械検証

## 構成

| パス | 内容 |
|---|---|
| `docs/` | 要件定義書 / 設計書 / 運用手順 |
| `research/` | 動画調査パイプライン（生データ・分類スクリプト・レビューMD） |
| `site/` | Astro + TypeScript 実装本体 |
| `site/src/data/` | バージョン・動画データ（JSON、運用で触るのは research 側） |
| `.github/workflows/deploy.yml` | GitHub Pages 自動デプロイ |

## クイックスタート

```bash
cd site
npm install
npm run dev      # http://localhost:4321 で確認
npm run build    # データ検証込みのビルド
```

技術スタック: Astro / TypeScript / Zod / GitHub Pages（ランニングコスト 0 円）
